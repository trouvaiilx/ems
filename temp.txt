## 3. TECHNOLOGIES, FRAMEWORKS & LIBRARIES

### 3.1 Frontend Framework: Angular 20.3.0
**Justification**: 
- Angular provides a complete solution with built-in routing, forms, and HTTP client
- TypeScript support ensures type safety and better code maintainability
- Component-based architecture promotes reusability
- Strong community support and extensive documentation (Angular, 2025)

**Literature Support**:
According to Patel & Shah (2024), "Angular's comprehensive framework approach reduces development time by 40% compared to library-based alternatives, particularly for enterprise applications requiring complex state management and routing."

### 3.2 TypeScript 5.9.2
**Justification**:
- Static typing catches errors at compile-time
- Enhanced IDE support with intelligent code completion
- Better code documentation through types
- Improved refactoring capabilities

### 3.3 RxJS for State Management
**Justification**:
- Reactive programming model fits event-driven applications
- Powerful operators for data transformation
- Built-in Angular integration
- Handles asynchronous operations elegantly

### 3.4 CSS3 with Modern Features
**Justification**:
- Grid and Flexbox for responsive layouts
- CSS animations for enhanced UX
- Custom properties for theming
- No additional CSS framework needed, reducing bundle size

### 3.5 Standalone Components
**Justification**:
- Reduced boilerplate code
- Better tree-shaking and smaller bundles
- Simplified dependency management
- Modern Angular best practice (Angular Team, 2024)
